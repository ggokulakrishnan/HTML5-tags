01. tag: History of HTML
Description:

(a) HTML stands for Hyper Text Markup Language
(b) HTML is a Markup language rather than a programming language
(c) Hyper-Text: means "Text within Text", and that text has a link within it
(d) HyperText is way to link two or more web-pages
(e) Markup: is a computer language, used to apply layout and format to a text document
(f) HTML History: In the late 1980's, a physicist (Physics Person), Tim Berners-Lee who was a contractor at CERN (European Organization for Nuclear Research), proposed a system for CERN researchers
(g) In 1989, he proposed an internet based hypertext system
(h) Tim Berners-Lee is known as the father of HTML

02. tag: HTML Versions
Description:

(a) HTML v1.0: It was released in 1991, the bare-bone vesion (comprising only the basic or essential elements)
(b) HTML v2.0: It was released in 1995, the standard language for all web-page design. It had features like form-elements (text-box, button, radio-button etc.), file-upload
(c) HTML v3.2: It was released in Jan 1997, this version was published by W3C and had features for additional form-elements, tables, complex mathematical operations.
Official standard for any browser, today's morden browser understands this version
(d) HTML v4.01: It was released in Dec 1999, this most stable version of HTML. Provides support for CSS and other multi-media elements
(e) HTML v5.0: This is the new version of HTML, two major Organizations were involved

  • W3C (World Wide Web Consortium)
  • WHATWG (Web Hypertext Application Technology Working Group)
(f) HTML is a case-insensitive language, which means we can use tags either in lower-case or upper-case
(to adjust data inside the div use padding and to adjust the parent div itself use margin)
Note: It is recommended to write all tags in lower-case for consistency, readability, etc.

07. tag: HTML Building Blocks
Description:

(a) The basic building block of HTML are (i) TAG (ii) ATTRIBUTE (iii) ELEMENT

  1. Tag: to provide a meaning, surrounded by < >
  2. Attribute: provides extra information about the element
    1. Always applied to the starting tag
    2. Attribute contains two fields
      • name
      • value
  3. Element: Element is an individual component and everything written inside HTML is termed as Element
    4. (Example): <p style="color: red">The style is attribute of paragraph tag</p>

08. tag: HTML tag
Description:

(a) HTML tags are always written in lower-case (Example): <p> </p> <h1> </h1>

09. tag: HTML Attribute
Description:

(a) They are special words providing additional information to HTML element
(b) Attribute defines the behaviour of the HTML Element
(c) Attributes are always applied to the starting tag
(d) Attribute always have Key and Value pair
(e) Attribute "name" and "value" are case sensitive, they should be followed as per W3C recommendation (in lower-case)

10. tag: HTML Element
Description:

(a) A HTML consists of Start tag, End tag and Content between them
(b) Few HTML Elements do not have End tags, these tags are called as:

  • Empty Elements
  • Self Closing Elements
  • Void Elements
(Example:) <br> (line-break), <hr> (horizontal-line)

13. tag: Block-Level Elements
Description:

(a) They always starts with new line and tasks full width of the web-page from left to right
(Example:) h1 - h6; table; main; hr, li, div etc..
(b) All these tags take full width of the web-page

14. tag: In-Line Elements
Description:

(a) They do not start with new line and take the width as per the requirement
(b) They are mostly used with other elements
(Example:) button, label, time, span, br, etc..

(d) Example: Hello HTML5

(h) HTML anchor tag apperance:

21. tag: HTML table
Description:

(a) table is used to display data in tabular format

  1. table: defines a table
    (a) table border, table caption, collapse-border
    (b) table cell-padding
    (c) table width
  2. tr: defines a table row
    (f) Styling HTML table even-odd cells
  3. th: defines header cells in a table
    4. (j) table th
  4. td: define cells in a table
    5. (g) table td
  5. caption: defines table caption
  6. colgroup: group of one or more columns in a table for formatting of columns
    (d) table col-span
    (e) table row-span
  7. col: used with element (colgroup), to specify column properties of the table
  8. tbody: used to group the body contents of the table
  9. tr: defines a table row
    (g) table body, tr, td
  10. template: tag is used to hold client side content and it will not be rendered at page_load, instantiated with Javascript
    (h) table template
  11. theader: used to group the header contents of the table
  12. tfooter: used to group the footer contents of the table
    (i) table tfoot
