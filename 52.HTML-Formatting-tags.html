strike text (strike)

monospaced font (tt)
each letter has the same width, fonts are known as variable-width fonts because different letters have different width. (for example: 'w' is wider than 'i')
Monospaced Font provides similar space among every letter
01. India is my country, I am a boy <tt>

Superscript Text (sup)
If we put a content with in <sup> then it will be a Superscript text. It displays half a character height above the other character
01. Rama is 1st in the class - <sup>

Subscript Text (sub)
If we put a content with in <sub> then it will be a Subscript text. It displays half a character height below the other character
01. Nitrogen is represented as N2 in the valency table - <sub>

Deleted Text (sub)
Anything that is put between <del> will be a deleted text.
01. The name of my country in Chennai India - <del>

Inserted Text (sub)
Anything that is put between <ins> will be an inserted text.
01. The name of my country is India - <ins>

Larger Text (big)
For a content if we want to put a larger text than rest of the content, then use the tag <big>
It increases one font size larger than the other font in the text
01. It is very cold is Chennai - <big>

Smaller Text (small)
For a content if we want to put a smaller text than rest of the content, then use the tag <small>
It decreases one font size smaller than the other font in the text
01. It is very cold is Chennai - <big>
