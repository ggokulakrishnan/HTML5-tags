(a) HTML <template> tag is used to hold client side content and it will not be rendered at page_load
(b) This tag can be instantiated at runtime using Javascript
(c) The content template will not be instantiated until rendered by Javascript
(d) During page_load the browser processes the content and ensures it is valid but it will not be rendered
(e) The <template> tag can be placed anywhere inside of HTML element
  • (head) <head>,
  • (body) <body>,
  • (frameset) <frameset>,
  • (table) <table>
(f) The <template> tag is newly added element in HTML5
